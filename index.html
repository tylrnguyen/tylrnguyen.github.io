Projects

SafeContractors

SafeContractors was built for the Civic Hacks 2025 economic track. Our goal was to provide a reliable database for people in the Boston Area to background check contractors. The database scraped with SQL was stored in MariaDB, and a FAST API was created to fetch the past contracting projects, amount paid, status, and an brief summary detailing whether the contractor is worth hiring.

Received award for Best Overall Hack and Best Scam Protection

GitHub DevPost

Automated Hackathon Judging System

As a program ambassador for Boston University's Technology Innovation Scholar Program started by former BU Engineering Dean Ken Lutchen, I had the privilege of being apart of the Technical Team for Boston University Hacks High School (BUHHS).

Prior to the Hackathon, I developed a judging system using React and MongoDB to create a software to automate scoring. In past years, parsing through judge scores took over 30 minutes, which significantly hindered the flow of the hackathon and was stressful for organizers.

Additionally, through this experience, I assisted in organizing workshops and the keynote speaker for the event, which was Steve Dertien - PTC's current EVP.

Morse Code Translator

The morse code translator was an interactive decoder through an FPGA board, aimed for users to increase their fluidity in the morse code language. The project was built using Verilog to interpret user inputs as dots and dashes, which were then displayed through visual graphic array adapted from an open source ASCII-ROM.

GitHub Live Demo

Physical State Tracking Wearable

With a team of 3, created a vest wearable equipped with LEDs connected to Arduino circuit with accelerometer and MOSFET to trigger LEDs. Designed for senior citizens to alert caregivers of crisis.

Award 2nd Best Wearable amongst 8 teams

Antidote

Antidote was made for the 2023 Lutron Lighting Competition hosted by Boston University Singh Imagineering Lab. Meant for those with seasonal depression, we created a window that emulated different times of day. To create this, we used an Arduino Uno and coded a C program to change the colors of specific LEDs.

Awarded competition's Most Innovative Entry
